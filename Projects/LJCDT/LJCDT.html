    Vision of the Solution

    Johnson County Leadership envisioned an automated mobile web-based system that will allow a single-person or more to navigate the historical tour/scavenger hunt more easily, safely, and raise the enjoyment level of the whole process.

    During my senior project, I learned the SDLC from start to finish. I was able to work with a client as much as possible due to contrast with Covid-19. I gained the core understanding of Planning, Analysis, Design, and Implementation in regards to build software. I also gained additional experience by doing Maintenance on a previous students project. The Full Stack Software Development process was experience based on the actions I took. It started from the beginning creating all the documentation needed, gathering all the requirements to design the database, to even designing a prototype in AdobeXD before tackling the coding process. During, this process I quickly self taught myself how to use Adobe XD, alongside how to implement Google Maps API, Text-to-Speech Google API, SweetAlert2, and Bootstrap while making part of the software system mobile responsive.
